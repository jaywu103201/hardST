## rand() 
```
亂數，在 C 語言中提供了一個 rand() 函數可以讓我們取得一個亂數


整數變數=rand();
```
```
include 它的標頭檔 stdlib.h

0 到 RAND_MAX  整數值



搭配使用取餘數符號 % 來做運算
0-9 的變數，可以寫成 a=rand() % 10


取 1-10 的亂數 a=(rand() % 10) +1

取 1-100 的亂數 a=(rand() % 100) +1

取 100-1000 的亂數 a=(rand() % 901) +100
```
