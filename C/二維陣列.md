## 二維陣列的定義
```
二維陣列定義的一般形式是：

    類型說明符 陣列名稱[常量運算式1][常量運算式2]
    
    
    運算式1表示第一維下標的長度，常量運算式2 表示第二維下標的長度。
    
       int a[3][4];
       一個三行四列的陣列，陣列名稱為a
       類型為整型
       標變數共有3×4
       
    a[0][0],a[0][1],a[0][2],a[0][3]

    a[1][0],a[1][1],a[1][2],a[1][3]

    a[2][0],a[2][1],a[2][2],a[2][3]
       
```
### 二維陣列的初始化
```
例如對陣列a[5][3]：

1)   按行分段給予值可寫為:

        int a[5][3]={ {80,75,92},{61,65,71},{59,63,70},{85,87,90},{76,77,85} };

2)按行連續給予值可寫為:

        int a[5][3]={ 80,75,92,61,65,71,59,63,70,85,87,90,76,77,85};

    這兩種給予初值的結果是完全相同的。
```
