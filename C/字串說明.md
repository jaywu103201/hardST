### 字串說明
```
字串就是一段文字，我們可以用一對雙引號 ( " ") 把該段文字夾起來

scanf 及 printf 讀入及印出一個字串則是使用 %s



 C 語言中並沒有字串的變數型態，而是用字元的陣列來儲存一個字串
```
```
char s[20], t[20]="LSSH"; 
scanf("%s", s);

printf("%s", s);
```
```
hello world
hello
--------------------------------
Process exited after 5.556 seconds with return value 0
請按任意鍵繼續 . . .



```
```
  s 是一個字串 
  t 長度為 20 個字元 
  字串後面都有一個 '\0' 
 "LSSH" 字串，事實上總共用了 5 Bytes(因為字串後面要加\0 表示結束)
```

### gets() 與 puts()


```
canf 讀取字串時，遇到空白字元便會結束，例如輸入 "Hello! LSSH!"    字串只有 "Hello!"    

gets 、puts (也是定義在 stdio.h)，
gets 的功能是讀入字串，而 puts 則是印出字串：
```

```
Enter your name: hello world
Your name is: hello world

--------------------------------
Process exited after 8.653 seconds with return value 0
請按任意鍵繼續 . . .



```


```
gets，會連換行一起讀進來，也就是字串後面會多一個 '\n' 字元

C 語言中的字串是字元的陣列
如果我們要知道第 N 個字元，就是用 s[N-1]的方式取得。
```

### string.h
```
strcpy(s1, s2)    將 s2 的內容複製到 s1

strcmp(s1, s2)    比較 s1、s2 的內容，如果相等傳回 0

strcat(s1, s2)    將 s2 串接到 s1 後面

strstr(s1, s2)    傳回 s2 字串在 s1字串中第一次出現的位置

strlen(s1)    傳回 s1 的長度(不含 '\0' 字元)

strrev(s1)    將 s1 字串倒置

https://skylinelimit.blogspot.com/2018/02/c-2.html
```


```
#include <stdio.h>
#include <string.h>
 
int main()
{
   char src[40];
   char dest[100];
  
   memset(dest, '\0', sizeof(dest));
   strcpy(src, "This is runoob.com");
   strcpy(dest, src);
 
   printf("最终的目标字符串： %s\n", dest);
   
   return(0);
}
```
