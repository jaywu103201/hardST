 ## 條件式
 ```
 判斷時要用的句型，比較兩個數值的關係
 
 條件式本身還不是一行指令，所以後面是不用加分號 ;
 ```
 ```
 < 小於

> 大於

== 等於 (注意，是兩個等號)

<= 小於或等於

>= 大於或等於

!= 不等於
 ```
 
 ```
 有兩個以上的條件，則可以使用 && (And 且)及 || (Or 或)將之連起來
 
 變數 a 大於10 且小於 100： (a>10) && (a<100)

變數 a 等於b 或等於 c： (a==b) || (a==c)


!，則代表否定(Not) 

例如：

!( a>9) 和 a<=9 是一樣
 ```
 if (條件式) 指令一;

上面的語法是如果符合該條件，則執行指令一



if (條件式) {

    指令一;

    指令二;

    指令三;

}

else {

    指令四;

    指令五;

    指令六;

}
 ```
