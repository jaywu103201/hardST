## 運算符號
```
+：加法
- ：減法
*：乘法
/ ：除法
%：餘數，只適用於整數型態



乘除後加減，有括號則先做
```

```
  - (單元的正負號)   ++   --           由 右 至 左
  *    /     %                        由 左 至 右
   +      -                           由 左 至 右
   =  +=  -=  *=  /=                  由 右 至 左
```
```
變數  雙目運算符=運算式

變數=變數 運算符 運算式

a+=5         等價於a=a+5

x*+=y+7    等價於x*=x*(y+7)

r%=p       等價於r=r%p

```

```
  ++i ;  // i自增1後再參與其他運算。

  --i;    //i自減1後再參與其他運算。

  i++ ;  //  i參與運算後，i的值再自增1。

  i-- ;   //i參與運算後，i的值再自減1。

```
```
#include<stdio.h>

main(){

  int i=8;

  printf("%d\n",++i);

  printf("%d\n",--i);

  printf("%d\n",i++);

  printf("%d\n",i--);

  printf("%d\n",-i++);

  printf("%d\n",-i--);

 }
```
```
 i的初值為8，
 第2行i加1後輸出故為9；
 第3行減1後輸出故為8；
 第4行輸出i為8之後再加1(為9)；
 第5行輸出i為9之後再減1(為8) ；
 第6行輸出-8之後再加1(為9)，
 第7行輸出-9之後再減1(為8)
```
